# Setup 

1. Navigate to ./TestServer

2. Setup Python3 environment:

    #### Linux:
    `python3 -m venv venv`

    #### Windows:
    `py -3 -m venv venv`

3. Activate the environment

    #### Linux:
    `. venv/bin/activate`

    #### Windows:
    `venv\Scripts\activate`

4. Install dependencies (Flask, SocketIO)

    `pip install Flask`

    `pip install flask_socketio`

# Usage

`python3 ./server.py`

Navigate to `http://127.0.0.1:5000`
